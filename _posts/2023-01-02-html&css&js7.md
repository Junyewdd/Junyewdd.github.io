내용
1. data-text는 사용자 정의 텍스트(속성)이다 (사용자 정의는 data-로 시작)
2. inline-block끼리 수직 중앙 맞출 때: vertical-align: middle;
3. html에서 id이름 중복되면 오류남
4. font는 before after에는 사용할 수 없지만 label에는 사용가능
5. active 상태 = 마우스 눌렀다가 떼지 않은 상태
6. 크기 다른 글 두 줄 만들 때 따로 만들 필요 없음. <h2>~~<span><span><h2>이렇게 해서 css에서 수정하기
7. <input type="text" placehoder="~~~">, <input type="submit" value="~~~">
8. background 가득 채우려면 'background-attachment: fixed'
9. input 요소는 속성 선택 가능 -> 선택자 사용할 필요 없음!
10. placeholder은 css3부터 생겼기 때문에 가상클래스 의미하는 : 한 번 더 써야 함(input[type=text]::placeholder)
11. padding으로 박스 height조절 하는 경우 있음
