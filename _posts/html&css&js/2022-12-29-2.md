내용<p>
  HTML 핵심 태그 사용법: 
  1. 제목 표시 : h1~h6
  2. 문단, 줄 바꿈 : p, br
  3. 글자 굵게 표시 : b, strong
  4. 글자 기울림 표시 : em, l
  5. 글자 밑줄 표시 : u
  6. 글자 취소선 표시 : s, del
  7. 큰 or 작은 글자 표시 : big, small
  8. 위 첨자, 아래 첨자 : sup, sub
  9. 형광펜 배경색 글자 : mark
  10. 수평선 넣기 : hr
  
  무의미한 한글 텍스트 - 한글입숨
  링크와 이미지 중첩 : a > img <a href = ""><img src=""></a>
  
  css 적용 우선순위
  1. !important style
  2. Inline style
  3. ID selector style
  4.Class selector style
  5. tag selector style
