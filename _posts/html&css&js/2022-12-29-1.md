#내용
visual studio 단축키
1. Alt+SHIFT 누르면 여러 줄 선택 가능
2. Alt : 다중선택
3. Alt+F12 누르면 HTML 내용에 부합하는 CSS 내용으로 이동 가능
4. Ctrl+, 누르면 visual studio code 설정창 열림
5. Alt+w 누르면 부모요소로 묶고싶은 곳 블록으로 지정 가능(by Htmltagwrap 확장)
